
ETL Airflow: Extração de Dados de Endereçamento Postal (CEP) via ViaCEP

Novo projeto no GitHub!

Acabei de construir um pipeline de ETL no Apache Airflow focado na extração de dados públicos de CEPs via API ViaCEP.

O objetivo principal foi demonstrar a criação de componentes Airflow:

1.  CepHook: Gerencia a conexão com a API e o tratamento de erros.
2.  CepOperator: Extrai uma lista de mais de 50 CEPs, salvando os resultados em um CSV.
